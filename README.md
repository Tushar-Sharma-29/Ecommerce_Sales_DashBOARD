# 🛒 E-Commerce Sales & Performance Dashboard

## 📊 Project Overview
A Power BI dashboard analyzing **3,510+ orders** across 2020–2024 
with **$122M net sales** and **$12.2M profit** to track business 
growth and performance.

## 🎯 Key Insights
- **Top Product:** Apple iPhone 14 — $21.4M in sales
- **Lowest Product:** Colgate Toothpaste — $0.02M in sales
- **Peak Sales:** $0.65M (2023)
- **Highest Discounted Category:** Electronics ($5M+)
- **Total Orders:** 3,510+ across 4 years

## 📌 Dashboard Pages
1. **Product Performance** — Top 5 & Bottom 5 by Sales, Units, Profit
2. **Sales Trend** — 4-year daily trend analysis (2020–2024)
3. **Geographic Analysis** — City-wise sales map across India
4. **Discount Analysis** — Discount breakdown across 8 categories
5. **Period Comparison** — Dual date filter for custom time range comparison
